# 98. Validate Binary Search Tree

### Question Link

{% embed url="https://leetcode.com/problems/validate-binary-search-tree" %}

### Solution

A valid BST has its in-order traversal sorted.

```python
class Solution:
    def isValidBST(self, root: Optional[TreeNode]) -> bool:
        stack = []
        prev = float("-inf")
        while root or stack:
            while root:
                stack.append(root)
                root = root.left
            root = stack.pop()
            if prev < root.val:
                prev = root.val
            else:
                return False
            root = root.right
        return True
```
