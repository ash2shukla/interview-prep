### Solution

Its same as trying to find the intersection point of two linked lists.

![](../../.gitbook/assets/image.png)

```python
def solve(p, q):
    temp_p = p
    temp_q = q
    while  p is not q:
        p = temp_q if p is None else p.parent
        q = temp_p if q is None else q.parent
    return q
```
