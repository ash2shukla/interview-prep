# 814. Binary Tree Pruning

{% embed url="https://leetcode.com/problems/binary-tree-pruning" %}

### Solution

If some of children is > 1 then we can leave the node be else we need to remove it by setting it to node.

```python
class Solution:
    def pruneTree(self, root: Optional[TreeNode]) -> Optional[TreeNode]:
        def dfs(node):
            if node is None:
                return 0
            else:
                left_val = dfs(node.left)
                right_val = dfs(node.right)
                if left_val == 0:
                    node.left = None
                
                if right_val == 0:
                    node.right = None

                return node.val + left_val + right_val
        
        if dfs(root) == 0:
            return None
        else:
            return root
```

Or we can just use boolean operations.

```python
class Solution:
    def pruneTree(self, root: Optional[TreeNode]) -> Optional[TreeNode]:
        def dfs(node):
            if node is None:
                return False
            else:
                left = dfs(node.left)
                right = dfs(node.right)
                if not left:
                    node.left = None
                
                if not right:
                    node.right = None

                return node.val or left or right
        
        if dfs(root):
            return root
```
